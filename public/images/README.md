# Images Folder

Place your 6 nature images in this folder with the following filenames:

1. `mountain-range.jpg` - Stark black and white mountain range
2. `coastal-cliff.jpg` - Dramatic coastal landscape with cliff and ocean
3. `bull-pasture.jpg` - Bull in green pasture
4. `mountain-mist.jpg` - Mountain peak emerging from clouds
5. `misty-forest.jpg` - Misty forest with autumnal colors
6. `tropical-island.jpg` - Crescent-shaped tropical island

**Note:** The images can be in JPG, PNG, or any web-compatible format. Just make sure the filenames match exactly (you can rename your files to match these names).
